# Task Packet

- Scope: 搭建 AIGC 文本检测实验原型、实验协议和 demo。
- Files to read: 课程实验报告模板、AIGC 检测相关论文与数据集页面。
- Files allowed to edit: `aigc-detection-research/`。
- Required skills: paper-orchestration, brainstorming-research, experiment-results-planning, nature-academic-search。
- Evidence/data inputs: arXiv/Nature 文献、课程报告模板、样例 CSV。
- Required artifacts: 研究方向文档、实验协议、baseline 代码、demo、烟测结果。
- Rejection checks: 不把样例数据性能当真实结论；不生成正文冒充论文成稿。
- Validation commands: `python src/aigc_detector/train.py ...` and `python src/aigc_detector/predict.py ...`。
