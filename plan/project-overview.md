# 项目概览

## 基本信息

- 课程：网络与信息安全
- 主题：AIGC 生成与检测
- 推荐题目：面向跨域与轻度改写攻击的轻量级可解释 AIGC 文本检测方法研究
- 当前阶段：S0/S3，完成方向收敛、baseline 原型与实验协议初稿
- 输出目标：课程实验报告；后续可扩展为专利交底或会议短文

## 研究目标

构建一个可复现的 AIGC 文本检测实验系统，重点关注跨域泛化、改写攻击鲁棒性和误报风险控制，而不是只报告单一准确率。

## 预期贡献

1. 轻量级混合特征检测 baseline。
2. 面向安全场景的跨域和扰动评测协议。
3. 带风险分级与解释输出的网页 demo。

## 课程报告映射

- 实验简介及目标：写研究背景、威胁模型、系统目标。
- 实验原理：写 TF-IDF、字符 n-gram、风格统计、逻辑回归、鲁棒性评测。
- 实验过程及结果：写数据、训练、评测、demo 截图和结果表。
- 实验中的问题及解决：写中文路径、编码、数据偏差、误报风险、改写攻击。
- 实验总结：写局限和后续论文/专利扩展。
